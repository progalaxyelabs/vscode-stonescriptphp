# Change Log

All notable changes to the "stonescriptphp-snippets" extension will be documented in this file.

## [1.0.0] - 2024-12-01

### Added
- Initial release of StoneScriptPHP Snippets extension
- Route snippets:
  - GET route with BaseRoute
  - POST route with validation
  - Route with contract implementation
  - Health check route
- Model snippets:
  - Basic model class
  - Model methods (create, get, update, delete)
- Contract snippets:
  - Basic contract interface
  - Route contract
  - Service contract
  - Repository contract
- DTO snippets:
  - Request DTO
  - Response DTO
  - Generic DTO
  - Validated DTO
  - Array and nullable property snippets
